# Planning Poker
![CI](https://github.com/philipphermes/planning-poker/actions/workflows/ci.yml/badge.svg)

## Development

Run the dev server:
```sh
yarn dev
```

## Drizzle

Generate Migration:
```sh
npx drizzle-kit generate
```

Migrate:
```sh
npx drizzle-kit migrate
```

Studio:
```sh
yarn drizzle-kit studio --port=3000
```
