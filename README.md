# Planning Poker
![CI](https://github.com/philipphermes/planning-poker/actions/workflows/ci.yml/badge.svg)

## Development

Run the dev server:
```shellscript
npm run dev
```

## Drizzle

Generate Migration:
```sh
npx drizzle-kit generate
```

Migrate:
```sh
npx drizzle-kit migrate
```

Studio:
```sh
npm run drizzle-kit studio --port=3000
```
